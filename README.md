# 🎵 dangerous_data_mj_statistics

Applied Statistics • Excel Analytics • MBA Learning Project • Michael Jackson Dataset

Statistical analysis applied to a real Michael Jackson dataset.

Built as part of my MBA in Data Science, AI and Analytics (USP/ESALQ), this repository translates statistical theory into practical analytical workflows using real music industry data.

---

## 📌 About the Project

This repository documents my MBA learning journey by applying statistical concepts to a real Michael Jackson dataset.

Instead of relying on synthetic datasets or abstract textbook examples, every concept is explored through real music industry data — including singles, albums, chart performance, sales figures and release history.

The project evolves progressively from Excel-based analysis toward Python implementation and Business Intelligence visualization.

No synthetic datasets.

No generic examples.

Only contextualized analytics using real musical data.

---

## 🧠 Project Philosophy

The objective is not only statistical correctness.

This project is designed to:

- reinforce MBA learning;
- transform abstract concepts into contextualized analysis;
- document analytical growth from Excel to Python and BI;
- build a practical analytics portfolio using meaningful datasets;
- make statistical reasoning tangible through familiar data.

## 📊 Dataset

### Main Dataset

**MJ_Dataset_Oficial_v2.xlsx**

| Variable | Type | Description |
|---|---|---|
| Single | Qualitative (nominal) | Song title |
| Album | Qualitative (nominal) | Source album |
| Year | Quantitative (discrete) | Release year |
| Peak EUA | Quantitative (discrete) | Best Billboard Hot 100 position |
| Sem. Chart EUA | Quantitative (discrete) | Weeks on Billboard Hot 100 |
| Vendas Album (mi) | Quantitative (continuous) | Worldwide album sales (millions) |
| Tem Clipe | Qualitative (binary) | Official music video (Yes/No) |
| Gênero Musical | Qualitative (nominal) | Musical genre |
| Duração (min) | Quantitative (continuous) | Single duration in minutes |
| Grammys | Quantitative (discrete) | Grammy awards for the album |

**26 singles analyzed • 6 studio albums • catalog coverage spanning 1979–2025 references**

## 🔍 Analytical Questions This Project Will Explore

- Do MJ singles remain more than 15 weeks on chart?

- Does Billboard peak position correlate with weeks on chart?

- Does having a music video affect chart performance?

- Is *Thriller* statistically identifiable as an outlier in album sales?

- Does musical genre influence chart success?

- Can chart longevity be partially predicted from ranking performance?
